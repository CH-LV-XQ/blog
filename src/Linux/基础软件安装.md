# 基础软件安装

## rz 命令
作用：XShell 用于文件上传功能。
### 安装
```shell
# CentOS
sudo yum install lrzsz

# Ubuntu
sudo apt-get update
sudo apt-get install lrzsz
```
### 检查rz命令的语法
确保你正确使用了rz命令。基本的用法是直接在命令行中输入rz，然后按Enter键，它会提示你选择文件进行接收。
